# ArchWiz Studio BETA v0.0.2
**Stabilized Canvas & Streamlined Onboarding.**

This release focuses on making the Variation Canvas more robust and simplifying the initial user journey with a faster, professional onboarding flow and real-time pricing sync.

### What's New
*   **Streamlined 5-Step Onboarding**: Replaced the complex 11-step flow with a tight 5-step journey. Users now reach the product's core value in seconds rather than minutes.
*   **Integrated Interactive Demo**: Merged style preferences directly into the demo. Pick a single style and see an instant AI render—no more redundant configuration steps.
*   **Lucide Iconography**: Replaced legacy emojis with high-quality Lucide icons across all onboarding screens for a premium, professional aesthetic.
*   **Real-Time Pricing Sync**: The app now pulls live pricing tiers directly from the cloud, ensuring you always see the latest plans and credit allocations matching the website.
*   **Clear Canvas Tool**: Added a dedicated "Clear Canvas" button to the Variation Canvas for quick workspace resets.

### Bug Fixes
*   **Stuck Node Selection**: Fixed a persistent issue where nodes would remain "stuck" in a selected state after app reloads.
*   **Interaction Race Conditions**: Refactored canvas state management to eliminate UI glitches during rapid node dragging or panning.
*   **Canvas Snap-to-Grid**: Enabled 15px grid snapping to keep your architectural node graphs organized and aligned.

### Security & Privacy
*   **Authenticated API Proxying**: Transitioned all generation and chat requests to a Supabase-authenticated proxy for enhanced data security.
