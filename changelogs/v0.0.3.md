# ArchWiz Studio BETA v0.0.3
**Official Windows Launch & Enhanced Creative Freedom.**

This landmark release introduces **Windows support for the first time**, bringing the full ArchWiz Studio experience to a new platform. We've refined the cross-platform experience with a custom titlebar, improved installer, and expanded the flexibility of the architectural generation engine.

### What's New
*   **Custom Branded Titlebar**: A fully custom, draggable titlebar with "ArchWiz Studio" branding and native-like window controls for Windows.
*   **Windows Native Upscaling**: Introduced high-performance, on-device upscaling for Windows using ONNX Runtime and DirectML, matching our macOS CoreML performance.
*   **Image-less Arch Generation**: The Generator Node now supports starting renders directly from a style selection without requiring a reference image.
*   **Smart Onboarding**: Returning subscribers now bypass the paywall automatically with a personalized "Welcome Back" flow.
*   **Unified Export Workflow**: Simplified the UI by unifying "Export to Mac" and "Save to PC" into a single "Export & Save" action.

### Improvements
*   **Model Management**: Added platform-aware filtering to ensure only compatible models (ONNX for Windows, CoreML for Mac) are visible in settings.
*   **Visual Polish**: Overhauled dropdown aesthetics, fixed handle scaling alignment on hover, and added platform-specific application icons.
*   **Installer Logic**: Refined the Windows installer to prefer standard installation paths (Program Files) and provide a guided setup experience.

### Bug Fixes
*   Fixed a focus issue where the sidebar prompt textarea was occasionally unclickable on first launch.
*   Resolved an issue where MetalFX was incorrectly visible as an option on Windows systems.
*   Fixed a validation alert that incorrectly blocked architectural renders when no images were connected.

### Security & Privacy
*   Enhanced subscription validation logic to securely check active status directly through our Supabase integration during onboarding.
